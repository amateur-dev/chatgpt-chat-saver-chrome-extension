ChatGPT Chat Saver Extension Test

Testing Instructions

  1. Load the extension in Chrome (chrome://extensions/)
  2. Navigate to chat.openai.com
  3. Start a conversation with ChatGPT
  4. Look for the green "Save as PDF" button
  5. Click it to test PDF generation

Note: Make sure you've downloaded the required libraries (jsPDF and html2canvas) using the setup.sh script or manually.

Sample Conversation Format

This shows the type of content the extension should capture:

User: Can you write a simple Python function to calculate fibonacci numbers?
ChatGPT: Here's a simple Python function to calculate Fibonacci numbers:
def fibonacci(n): if n <= 1: return n else: return fibonacci(n-1) + fibonacci(n-2) # Example usage for i in range(10): print(f"F({i}) = {fibonacci(i)}")

This function uses recursion to calculate the nth Fibonacci number. The Fibonacci sequence starts with 0 and 1, and each subsequent number is the sum of the two preceding numbers.

User: That's helpful! Can you also show me the mathematical formula?
ChatGPT: The mathematical formula for Fibonacci numbers is:

F(n) = F(n-1) + F(n-2)

With base cases: F(0) = 0 and F(1) = 1
